# Артём Кухарев — Персональный сайт-резюме

Минималистичный, адаптивный одностраничный сайт-портфолио.

## 🛠 Стек

- **HTML5** — семантическая разметка
- **CSS** — кастомные свойства, oklch, `clamp()`, scroll-driven анимации
- **Vanilla JS** — переключение тёмной темы, плавная навигация
- **Google Fonts** — Instrument Serif + DM Sans

## 🚀 Деплой

Сайт размещён на **GitHub Pages** и доступен по адресу:

```
https://<username>.github.io/ResumeWeb/
```

### Локальный запуск

Просто откройте `index.html` в браузере — сайт полностью автономный и не требует сборки.

## 📁 Структура

```
ResumeWeb/
├── index.html     # Весь сайт (HTML + CSS + JS)
├── .nojekyll      # Отключение Jekyll для GitHub Pages
└── README.md
```

## ✨ Возможности

- 🌗 Светлая / тёмная тема (с определением системных настроек)
- 📱 Адаптивный дизайн (mobile-first)
- ⚡ Scroll-driven анимации (CSS `animation-timeline`)
- 🎨 Современная палитра oklch
- ♿ Поддержка `prefers-reduced-motion`
- 🔍 SEO: Open Graph, Twitter Card, favicon
